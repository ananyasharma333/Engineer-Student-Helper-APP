# Engineering Student Super App

A comprehensive platform designed for engineering students, featuring internship tracking, coding resources, project management, and more.

## Project Structure

- `frontend/`: React + Vite application.
- `backend-java/`: Java Spring Boot backend (REST API).
- `backend/`: Node.js/Express backend.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Axios, Lucide-React.
- **Backend (Java)**: Spring Boot, MongoDB Atlas.
- **Backend (Node)**: Node.js, Express.

## Getting Started

### Prerequisites

- Java 17+
- Node.js 18+
- MongoDB Atlas Account

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Engineering-Student-Super-App.git
   ```

2. Setup Backend (Java):
   ```bash
   cd backend-java
   ./mvnw spring-boot:run
   ```

3. Setup Frontend:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Features

- **Internships**: Search and track latest internship opportunities.
- **Coding**: Access coding challenges and resources.
- **Projects**: Collaborate on and manage student projects.
- **Notes**: Share and access engineering study materials.
- **Resume**: AI-powered resume tips and tools.
